When your repo has a docs/ folder with github-pages enabled* in the repo settings,
    you can see this file as delivered through a web server
    by using the url: 
       https://dierk.github.io/WebEngineering-Template/Example.html
                <1>                      <2>                <3>       
    <1> User
    <2> Repository
    <3> File					
			
    * You enable github-pages in the repository settings - Options - Github Pages.
      Select the second option: master-branch / docs folder
